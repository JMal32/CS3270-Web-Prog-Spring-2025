Questions and Answers

Q: Your 2p game presumably is played on two "reef" boards. What are the names of the two reef-board image files your captured game-state is using?
A: b0.jpg and b2.jpg

Q: Find the two lines of code you would need to change at game-setup to ensure that your game used (for example) board0 and board1 (from the set of boards [board0 … board3] ). Show the HTML tags that would result in those two board-images being displayed in your browser, board0 on the Left, and board1 on the Right.
A: The HTML tags would be: 

<IMG SRC="game/reef/images/b0.jpg">
and 
<IMG SRC="game/reef/images/b1.jpg">
These would replace the current image tags in lines 557 and 559 that contain b0.jpg and b2.jpg respectively.

Q: You may notice that most of the Control_Panel action-links in Neville's IS page are deactivated (and action-images are grayed out), as opposed to the links and images in your own captured page. Explain how this was accomplished, then implement the same changes in your own page.
A: This is accomplished by: 1. Removing the href attribute from anchor tags to make them non-clickable text 2. Adding a CSS class like "disabled" to links and images 3. Setting CSS properties such as opacity:0.5 and cursor:not-allowed 4. For images, applying a grayscale filter with CSS To implement this, we would: - Add CSS rules in the style section - Convert action links to plain text or apply the disabled class - Apply grayscale or reduced opacity styling to the action images
